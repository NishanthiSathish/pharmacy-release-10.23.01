• The dose is entered as a number of a particular unit, which can be chosen from the drop down.
  • The system will only offer units which are applicable to the current product.
  • In the case of products with multiple active ingredients, the dose is expressed in terms of the total of all ingredients. So for example, the dose for Co-Amoxiclav infusion is expressed as the total of Co-Amoxiclav, not simply in terms of the amoxicillin component.
  • In the case of calculated doses, you can view and change the values used in the calculation by clicking on the [calculation] button. You can also apply a percentage adjustment to the value by clicking on the [Adjust] button.
Template Mode Only
  • Calculated doses can be specified by selecting a calculation parameter (such as 'per Kg') from the 'per' box. Leave this box blank for non-calculated doses.
  • If a calculated dose is specified, you may set rounding, capping and other options by clicking on the [options] hyperlink.

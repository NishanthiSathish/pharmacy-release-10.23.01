• The dose is entered as a number of a particular unit, which can be chosen from the drop down.
  • The system will only offer units which are applicable to the current product. Often, this will only be a single unit, in which case the unit cannot be changed. Note that simple liquids may be dosed in units of mass OR volume (eg, mL or mg).
  • In the case of calculated doses, you can view and change the values used in the calculation by clicking on the [calculation] button. You can also apply a percentage adjustment to the value by clicking on the [Adjust] button.
Template Mode Only
  • For doses specified in terms of mass, a calculated dose may be specified by selecting a calculation parameter (such as 'per Kg') from the 'per' box. Leave this box blank for non-calculated doses.
  • If a calculated dose is specified, you may set rounding, capping and other options by clicking on the [options] hyperlink.

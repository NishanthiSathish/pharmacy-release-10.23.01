• The Directions field is where the directions for administration should be entered. It is split into two parts; the top part of the field is where preset directions are entered, and the bottom part is for any further free text. Note that a preset direction MUST be entered, any extra free text is optional.
  • Preset Directions can be entered using the [...] button to the right of the field. This displays a list of appropriate coded directions from which one can be selected.
  • To clear a direction which you have selected, either select another one or use the [X] button to the right of the field
  • If required, free text can be entered in the box as required.
  • Note that any instructions relating to the supply of the medication, rather than the administration of it, should be entered into the "Dispensing Instructions" field
